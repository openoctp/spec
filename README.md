# OCTP Specification

**Open Contribution Trust Protocol — v0.1**

The OCTP specification defines a lightweight, 
machine-readable trust envelope that travels 
with every code contribution — communicating 
provenance, verification status, and human 
review level in a cryptographically verifiable format.

v0.1 — Community Draft. Open for review and discussion.
