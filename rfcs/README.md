# OCTP RFCs

RFC stands for Request for Comments.

An RFC is required for any significant change to the specification.
See GOVERNANCE.md for the full list of what requires an RFC.

## How to propose an RFC

1. Open an issue using the RFC Proposal template
2. The community discusses for a minimum of 14 days
3. A maintainer posts the decision with reasoning
4. Accepted RFCs are implemented via pull request
