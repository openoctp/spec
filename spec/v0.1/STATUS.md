# OCTP v0.1 — Status

**Current status: COMMUNITY DRAFT**

This version is open for community review and discussion.
It is not yet stable. Fields and structure may change based
on community feedback before v0.1 is marked stable.

## What this means

- Safe to experiment with and build prototypes against
- Not recommended for production use yet
- Your feedback directly shapes the stable version
- Open issues represent known gaps seeking community input

## Path to stable

v0.1 will be marked stable when:
- At least 3 real projects have piloted the spec
- No outstanding RFC proposals that would change the schema
- The reference tool implements the full spec
- The community has had 60 days to review
